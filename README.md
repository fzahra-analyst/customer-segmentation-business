# 📊 Customer Segmentation Business Analysis

## 📌 Project Overview

This project focuses on customer segmentation using data analysis and Machine Learning techniques.

The objective is to analyze customer behavior, identify different customer groups, and generate business insights that can help companies improve marketing strategies and customer targeting.

The project combines Python analysis and Power BI visualization to better understand customer patterns and segmentation.

---

# 🎯 Business Problem

Businesses often have thousands of customers with different behaviors and purchasing habits.

Without segmentation, companies may struggle to:

- Personalize marketing campaigns
- Understand customer behavior
- Identify high-value customers
- Improve customer retention
- Optimize business strategies

This project aims to solve these challenges through customer segmentation analysis.
